Exercise 2.1 – Bootstrap Folder Structure Explained

When you download Bootstrap from the official site, the ZIP file typically contains the following structure:

bootstrap/
├── css/
│   ├── bootstrap.min.css
│   ├── bootstrap.css
│   └── bootstrap.min.css.map
├── js/
│   ├── bootstrap.bundle.min.js
│   ├── bootstrap.min.js
│   └── bootstrap.bundle.min.js.map
└── icons/ (optional, if you downloaded Bootstrap Icons separately)

css/ Folder

  • Compiled CSS files that style your webpage.
  • bootstrap.min.css – Minified version for production use.
  • bootstrap.css – Unminified version, easier to read or customize.
  • *.map files – Help map minified CSS to source code (useful for debugging).

js/ Folder

  • JavaScript plugins that power components like modals and tooltips.
  • bootstrap.bundle.min.js – Includes Bootstrap and Popper (required for dropdowns/tooltips).
  • bootstrap.min.js – Bootstrap only (requires Popper to be included separately).
  • .map files – Help map minified JavaScript to the original source for debugging.

icons/ Folder (if included)

  • SVG icons from the Bootstrap Icons set.
  • You can use icons by linking this CSS file:
    • <link rel="stylesheet" href="icons/bootstrap-icons.css">
